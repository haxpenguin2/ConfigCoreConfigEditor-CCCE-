# ConfigCoreConfigEditor-CCCE-
# If you want to make your OWN plugins, contact my email in the README.

A simple linux app with PyQT5&6 that you can use to install compatibility modules to edit compatible text-based config files. WARNING: Some config editors may break config files and make sure to use the app with caution and make backups. Hopefully, compatibillity with other apps will be made by the community. Current support inclued Vim, I3, Picom, Bashrc, NetworkManager(nmcli) and MPV. email is jahomes.duytes@gmail.com. if you want to make plugins or your favorite services, contact me! i would really like a better programmer than me to make a plugin to edit GRUB with a gui.
